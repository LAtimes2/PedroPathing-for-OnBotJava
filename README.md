# PedroPathing for OnBotJava



This repository provides files that can be uploaded to OnBotJava on an FTC Control Hub to run the [PedroPathing](https://github.com/Pedro-Pathing) path follower library.



PedroPathing is usually used in Android Studio, but sometimes a team may want to use OnBotJava instead. The OnBotJava version is the same as the Android Studio version except that the Panels interface has been removed from Tuning (since I haven't figured out how to implement Panels in OnBotJava).



Instructions:

1. Upload 3 files from this directory to OnBotJava (PedroPathing.jar, Constants.java, Tuning.java)
2. Follow the instructions for PedroPathing, starting at [Constants](https://pedropathing.com/docs/pathing/constants)



This version of PedroPathing.jar is version 2.0.6. See the source\_code folder for the PedroPathing code in the jar file and instructions on how to rebuild the jar file for newer versions.
